Upgrade procedures

Warning

If you made any changes in application files, you must backup first before proceeding in the following upgrade procedures

  • Download the latest version for Frelancers Group application .
  • Replace your hosted application files with the new version
  • Adjust the folder permissions and database connection details
  • Go to upgrade script located in new application : Type your Application main URL followed by "/upgrade"
  • Follow the procedures there to upgrade your script
  • You done
